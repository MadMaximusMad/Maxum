# Glossary

**Capital** refers to accumulated value within the balance sheet available for deployment.

**Liquidity** refers to capital deployed into markets.

**Balance Sheet** represents the aggregate state of system capital and RFV.

**Risk-Free Value (RFV)** defines the stable asset backing of supply.

**Activity** represents all economic interactions within markets.

**Fees** convert activity into capital.

**Managed Liquidity** refers to system-deployed liquidity.

**Capital Efficiency** describes how effectively capital cycles through the system.

The terms defined here are operationalized across [System Model](../system/system-model.md), [System Flow Reference](system-flow-reference.md), and [Technical References](technical-references.md).
