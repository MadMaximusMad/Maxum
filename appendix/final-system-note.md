# Final System Note

**Maxum is a continuous capital system.**

It does not operate through isolated features or independent modules. It operates through the interaction of capital, liquidity, markets, activity, fees, and the balance sheet.

Each section of this documentation describes a different perspective of the same system.

{% hint style="success" %}
**Reading model**

The documentation should be read as a decomposition of one system, not as a catalog of separate features.
{% endhint %}

A first-pass reading should begin with [Overview](../README.md), [What is Maxum](../system/README.md), and [System Loop](../system/system-loop.md).
