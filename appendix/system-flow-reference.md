# System Flow Reference

**The system operates through a continuous loop:**

> Capital → Liquidity → Markets → Activity → Fees → Balance Sheet → Capital

Each component reinforces the others, forming a self-sustaining structure in which capital is continuously deployed, utilized, and expanded.

{% hint style="info" %}
**Reference use**

All major sections of the documentation describe a different point, mechanism, or control within this same loop.
{% endhint %}

For the full structural explanation, see [System Loop](../system/system-loop.md). For the operational path of capital, see [Flow Integration](../capital-flow/flow-integration.md).
