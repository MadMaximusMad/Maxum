# Technical References

Core system mechanics are defined by deterministic models. Automated market maker pricing follows the constant product invariant `x · y = k`. Fee-on-transfer logic ensures that value capture occurs only during liquidity interactions. Liquidity rewards are distributed through accumulator-based accounting. Supply is constrained by balance sheet RFV.

These technical references support [Fee System](../mechanisms/fee-system.md), [Liquidity System](../mechanisms/liquidity-system.md), and [Supply Model](../mechanisms/README.md).
