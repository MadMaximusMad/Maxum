# Capital Formation

**Capital formation defines how the system accumulates value.**

Capital enters the system through transaction-layer fee accumulation and direct capital deposits. Fee accumulation converts activity into capital in real time, while deposits increase the system’s capital base through external contributions.

All capital is consolidated within the balance sheet and does not remain idle. It is continuously prepared for deployment into liquidity. As capital formation increases, the system gains greater capacity to support markets, generate activity, and reinforce its own expansion.

{% hint style="info" %}
**Key property**

Capital formation increases when activity produces measurable output or when external capital is committed to the system.
{% endhint %}

Capital formed here is consolidated in [Balance Sheet](balance-sheet.md) and made productive in [Capital Deployment](capital-deployment.md). The transaction source of this process is described in [Revenue Layer](revenue-layer.md).
