# Flow Integration

**All capital flow layers operate as a single system.**

Capital formation increases available capital. That capital is deployed into liquidity. Liquidity enables markets. Markets generate activity. Activity produces revenue. Revenue expands the balance sheet. The balance sheet increases capital capacity.

This is not a sequence of independent steps. It is a continuous system in which each component reinforces the others.

{% hint style="success" %}
**System property**

Efficiency is defined by how effectively capital cycles through this loop and returns in an expanded form.
{% endhint %}
