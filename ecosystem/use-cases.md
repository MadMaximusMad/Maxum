# Use Cases

**Maxum operates in environments where activity is persistent and measurable.**

In trading environments, liquidity depth enables high-throughput execution and continuous fee generation. In prediction markets, recurring participation produces sustained transaction flow. In gaming economies and entertainment platforms, user interaction generates consistent activity cycles.

Across all environments, the defining requirement is that activity is measurable and capable of producing revenue.

{% hint style="info" %}
**Selection criterion**

The system is optimized for contexts in which usage is continuous and economically meaningful.
{% endhint %}

Use cases should be understood through [Activity Layer](../capital-flow/activity-layer.md), [Revenue Layer](../capital-flow/revenue-layer.md), and [Growth Model](README.md).
