# Parameter Control

**Governance modifies how capital flows through the system.**

Parameters controlled through governance directly influence system behavior. Adjustments to fee distribution alter how revenue is allocated. Changes to capital formation inputs affect how capital enters the system. Modifications to liquidity deployment strategies influence where and how capital is utilized.

These changes propagate through the system loop, affecting capital formation, liquidity distribution, activity generation, and balance sheet growth.

{% hint style="warning" %}
**Propagation**

Parameter changes do not remain local. They propagate through the full capital loop.
{% endhint %}

Governance decisions are therefore system-level adjustments, not feature updates.
