# Metrics & Transparency

Maxum provides visibility into system performance through a set of core metrics.

These include the capital base, liquidity under management, circulating supply, risk-free value, revenue generation, and market depth. Together, these metrics provide a comprehensive view of system health and growth.

By making these metrics accessible, Maxum ensures that system behavior is transparent and that participants can evaluate performance over time. This transparency strengthens trust and supports informed decision-making.
