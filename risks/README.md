# System Risk

**The system depends on continuous throughput.**

If activity declines, fee generation decreases, balance sheet growth slows, and capital deployment capacity is reduced. This weakens the reinforcing loop that drives system expansion.

The system does not rely on emissions to offset reduced activity. As a result, sustained inactivity directly impacts growth.

{% hint style="warning" %}
**System sensitivity**

The system weakens if throughput declines for a sustained period.
{% endhint %}

This risk is the inverse of the logic described in [System Loop](../system/system-loop.md), [Activity Layer](../capital-flow/activity-layer.md), and [Revenue Layer](../capital-flow/revenue-layer.md).
