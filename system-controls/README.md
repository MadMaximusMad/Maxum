# Stability Mechanisms

**Stability mechanisms preserve alignment between capital, supply, and liquidity.**

Maxum does not attempt to stabilize price directly. Instead, it maintains structural consistency between circulating supply and balance sheet capital.

Supply expansion is permitted only when capital enters the system. This ensures that all newly issued supply is backed by RFV.

When supply exceeds efficient levels relative to capital, the system deploys balance sheet capital to reduce circulating supply.

{% hint style="success" %}
**Structural outcome**

Stability emerges from constraint, not intervention.
{% endhint %}

This approach ensures that the system maintains coherence even under changing market conditions.

These controls depend on [Supply Model](../mechanisms/README.md), [Capital Expansion & Contraction](../mechanisms/capital-expansion-and-contraction.md), and the capital state defined in [Balance Sheet](../capital-flow/balance-sheet.md).
