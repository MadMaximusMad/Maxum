# What is Maxum

**Maxum is a stateful capital system.**

It is not a protocol, application, or financial product. It is an infrastructure layer in which capital is continuously formed, deployed, utilized, and reinforced through measurable market activity.

The system is balance sheet–driven. All capital is accumulated within the balance sheet, and this accumulated capital defines the system’s operational capacity. Supply issuance, liquidity deployment, and yield distribution are derived from this capital base rather than from external incentives.

Liquidity is not passively provided. It is deployed capital under system control, allocated into markets where it generates activity, revenue, and further capital formation.

{% hint style="warning" %}
**Interpretation**

Maxum should not be read as a collection of features. It should be read as a system in which capital transitions between states.
{% endhint %}

Maxum should therefore be understood as a continuous process rather than a static structure. That process is formalized in [System Model](system-model.md) and closed in [System Loop](system-loop.md).
