# System Loop

**Maxum operates as a closed-loop capital system.**

> Capital → Liquidity → Markets → Activity → Fees → Balance Sheet → Capital

Capital is deployed into liquidity. Liquidity enables markets. Markets produce activity. Activity generates fees. Fees expand the balance sheet. The balance sheet increases capital capacity. Capital is redeployed.

This loop is continuous and self-reinforcing.

Each component of the system exists to strengthen this loop. None of the components operate independently, and no part of the system generates value in isolation.

{% hint style="success" %}
**System property**

System growth is a function of throughput, not external inflows.
{% endhint %}

As activity increases, revenue increases. As revenue increases, capital increases. As capital increases, liquidity expands. Expanded liquidity enables further activity.

The loop begins with [Capital Formation](../capital-flow/README.md), becomes productive in [Market Layer](../capital-flow/market-layer.md), and returns to system state in [Balance Sheet](../capital-flow/balance-sheet.md).
